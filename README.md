# 數字轉換練習器

這個資料夾可以直接上傳到 GitHub repository 根目錄，作為 GitHub Pages 公開網站。

## 檔案

```text
index.html
number-converter.html
mapping.html
quiz.html
test.html
.nojekyll
```

`index.html` 是轉換器入口；上方導覽會連到獨立的對照表、自我測驗與限時測試頁面。

## GitHub Pages 設定

如果把這個資料夾內的檔案放在 repository 根目錄，GitHub Pages 請設定：

```text
Settings -> Pages -> Deploy from a branch -> main / root
```

公開網址通常會是：

```text
https://你的github帳號.github.io/repository名稱/
```
